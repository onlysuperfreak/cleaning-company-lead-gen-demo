# Bronx Fresh Cleaning Co. Landing Page

This is a fictional portfolio demo for a local cleaning company lead-generation landing page.

## Included Files

- `index.html` - page structure and landing page sections
- `styles.css` - mobile-first responsive styling
- `script.js` - demo form handling, CTA tracking, and owner dashboard updates
- `README.md` - project notes

## Demo Features

- Sticky header with phone number and quote CTA
- Hero section with conversion-focused copy
- Cleaning services and customer type sections
- Three cleaning package cards
- Placeholder image gallery
- Built-in ad tracking demo metrics
- Quote request form
- Business owner lead dashboard
- FAQ and fictional business disclaimer

## Notes

This project uses plain HTML, CSS, and JavaScript only. There are no frameworks, build tools, or external package dependencies.

Bronx Fresh Cleaning Co. is not a real business. This page is intended only as a portfolio concept for landing page design, lead capture, and local service marketing.
